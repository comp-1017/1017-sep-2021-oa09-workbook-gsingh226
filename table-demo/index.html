Table Demo

A table is a way of creating a structured grid of data. the tags we will be interested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the head row of the table
  • <tbody> : identifies the deta detail body of the table
  • <tfoot> : identifies the footer row of the table
  • <tr> : identifies the start of the a row on table
  • <th> : identifies the cell on the header row for the coloum title
  • <td> : identifies the cell on the detail row or footer row to hold content

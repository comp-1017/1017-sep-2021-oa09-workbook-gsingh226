Create navigational linkswith aweb site

there are several techniques to navigating within a website. The semantic tag for navigational is the <Nav> tag. you can navigate through your websiteby using relative addressing. For our examples we will unorder list tag (<ul>). our example will consist of:

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page (on page navigation)
  • going to a specific place onanother page

Navigational menus are placed in your < header> semantic area. They may also place in your < footer> area
